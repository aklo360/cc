## Feature Announcements (Thread System)

When an experiment ships, the brain posts a 4-tweet thread:

| Tweet | Content | Attachment |
|-------|---------|------------|
| 1 | Feature intro + highlights + link | Video trailer |
| 2 | Unique/fun detail about the feature | - |
| 3 | Technical explanation (commit-reveal, etc.) | - |
| 4 | CTA with link | - |

**Thread structure:**
- Generated by Claude during planning phase (Phase 1)
- First tweet is cross-posted to $CC community + all followers
- Tweets 2-4 are replies, forming a thread
- Thread content is stored in `plan.thread.tweet1-4`
